# 星舰熔炉 · 掉落合成 × 自走棋

科幻题材的抖音小游戏：**熔炉掉落合成（备战）→ 拖拽编队 / 升星 → 自走棋自动战斗 → 结算 → 下一波**，打穿五个星区、回到被封锁的家园「新伊甸」。

> **两个版本**
> - `cocos/` — **Cocos Creator 3.8 工程（TypeScript）**，面向抖音小游戏发布，**当前主力开发版本，已接入正式美术**。打开 / 预览 / 抖音构建见 `cocos/移植说明.md`。
> - 仓库根目录 — **web 原型（纯 JavaScript + matter.js）**，浏览器即开即玩，作为玩法快速验证与逻辑参照。两版玩法逻辑一一对应。

## 如何运行

**Cocos 版（推荐）**：Cocos Creator 3.8 打开 `cocos/` 工程 → ▶ 预览。首次需一次性初始化（matter.js 设为插件、挂 GameApp、设分辨率 480×840），详见 `cocos/移植说明.md`。

**web 原型**：双击 `index.html`；或 VS Code 装 `Live Server` 右键打开（改代码自动刷新）。另有单文件版 `星舰熔炉_单文件可玩.html`，方便发别人/手机试玩。

## 玩法循环

1. **备战（熔炉）**：移动瞄准、点击投放零件，相同的碰一起合成升级。合成链共 **12 级**（纳米芯片 → … → 母舰 → 彩蛋歼星旗舰）。到 **5 级「攻击无人机」及以上**的战舰带一个**阵营+舰种**标签（外圈颜色=阵营）。堆过「过载线」约 2 秒 → 熔炉过载，本局结束。
2. **编队部署**：点「编队部署」拖动战舰布阵——前排扛伤、后排受保护输出；**相同战舰（同级同星）叠一起升星**（★→★★→★★★，属性 ×1 / ×1.8 / ×3.2）。顶部实时显示激活羁绊，底部「待编区」放暂不上阵的战舰。
3. **自动战斗**：舰队按阵营/舰种羁绊自动开打，前排优先被攻击、吃「突击」减伤；暴击/护盾/再生/治疗/复活/溅射/召唤援军实时结算。必要时点「战术技·全体齐射」放群体爆发。
4. **结算**：胜利得金币、进下一波；每星区第 3 波是 **Boss**；失败可「重试本波」回熔炉重建。

对局中可随时点左上角 **🏠 主界面** 返回主菜单（保留大关进度，当前波次重新备战）。

## 四阵营 × 四舰种羁绊（2/4 档）

帝国（攻击·金橙）、异星（回血/复活·猩红）、机械（护盾/召唤·银蓝）、赛博（暴击/攻速·霓紫）；突击（前排减伤）、炮舰（增伤/溅射）、无人机（召唤僚机）、辅助（治疗）。凑同类 ×2 激活第一档、×4 更强档。数值见配套《羁绊与关卡数值表》。

> 升星取舍：两艘合成一艘会**减少该羁绊计数**（升星变强 vs 保持羁绊广度），是策略点。当前升星按「同级同星」合成，继承目标的阵营/舰种。

## 正式美术（已接入 Cocos 版）

美术用**即梦 AI**生成、程序化抠背景/裁切后接入，全套 tex 约 5MB，位于 `cocos/assets/resources/tex/`：

- **阵营战舰**：帝国用 12 级合成链原图；**异星 / 机械 / 赛博**各有 5~12 级共 8 艘**专属外观**（`ship_{阵营}_{等级}`）。战斗与编队里三个阵营的战舰各具辨识度。
- **五星区场景**：每个星区一套**背景 + Boss 立绘 + 杂兵模型**（`bg_s{1..5}` / `boss_s{1..5}` / `enemy_s{1..5}`）。打进不同星区，战斗背景、敌方杂兵、Boss 都会切换成对应主题（残骸带/海盗 → 机械坟场/机械哨兵/铁卫 → 虫巢/虫族/女皇 → 深渊/AI构造/零号意志 → 帝国终末/帝国护卫/万王之王）。
- **主界面 / 星图**：旗舰主视觉、归途星图背景。

渲染层 `cocos/assets/scripts/view/`：`Tex.ts` 负责加载与 `shipKey/bossKey/enemyKey/bgKey` 取图；`GameApp.ts` 按阵营 / 当前星区动态选图，加载完成前自动回退矢量画法。补图提示词见 `美术提示词_补充.md` 与 `4_星舰熔炉_战舰美术AI提示词.md`。

## 剧情

大寂灭后，你是曲率熔炉舰「星炉号」的末代舰队指挥官，从残骸中重建舰队，打穿海盗 → 机械 → 虫族 → 叛变AI → 帝国五个星区，回到家园「新伊甸」。剧情以**剧情卡**形式呈现（序章 / 每星区进场 / Boss 登场台词 / 通关结局，点击继续）。完整设定见 `剧情设定.md`；文案集中在 `js/story-data.js`（Cocos 版 `core/storyData.ts`）。

## 登录与主界面

启动进入**登录页**：输入「指挥官代号」（2~12字，即用户ID）或「游客进入」。每个代号独立存档（key 按 uid 隔离），记住上次登录、下次直达主界面。正式版把登录换成抖音 `tt.login`（openid 作 uid）即可，接口不变。

**主界面**三个入口：开始游戏（有进度续档 / 无进度播序章）、关卡选择、系统设置（音效开关 / 清除本账号存档 / 切换指挥官）。**关卡选择**是一张归途星图，五个星区节点按剧情排布，打通才点亮、未解锁显示「航道封锁」，可回跳重打（`maxLevel` 记录解锁上限）。静音键在右上角。

## 新手指引

首次游玩以**轻量提示卡**教核心三步：投放合成 → 编队拖拽 → 战术技，在对应阶段弹出、点任意处继续、做对后自动收起。按 uid 持久化，二次进入不再出现（系统设置里「清除存档」会连带重置指引，方便重看）。

## 经济与广告

金币来源：打赢波次（基础 + Boss 加成，随星区提升）。消费：**刷新队列**（换投放件凑合成路线）、**招募援军**（空降一艘可上阵战舰）。广告点位均为**主动点击换增益**（免费援军 / 胜利金币双倍 / 败北旗舰超载重打），原型用 3 秒模拟；正式版在 `ads.js`（Cocos `core/ads.ts` + `core/platform.ts`）换抖音 `tt.createRewardedVideoAd`。

## 音效

程序化合成音效（Web Audio API，零音频文件）共 15 种：投放、合成、升星、部署、开战、激光、爆炸、Boss 击毁、胜负、战术技、金币、拒绝、剧情卡、过载。右下 🔊 静音（偏好记住）。Cocos 真机无 WebAudio 时自动静音，正式版换真实音频文件只改 `audio.js`。

## 存档与防篡改

进度（星区/波次/金币/分数/最高合成）自动存档，再次打开续档；「重新开始」= 清档。防篡改两级：**本地**——存档带版本号 + 加盐校验和，手改即校验失败并重置；**服务器（接口已预留）**——HMAC 签名、客户端只验签，本地失败强制采用服务器档。抖音端本地存储换 `tt.setStorageSync`、网络走 `tt.request`，结构不变。

## 代码结构（模块化）

按「数据层→服务层→系统层→表现/交互→装配」分层，全局单命名空间 `SF`（web）/ 纯 TS 模块（Cocos `core/`，零 cc 依赖）。web 版加载顺序写在 `index.html` 和 `build.py` 两处。

```
js/                     web 原型（SF 命名空间）        cocos/assets/scripts/core/ 对应
  config.js             数值配置                        config.ts
  story-data.js         剧情文案                        storyData.ts
  state.js              运行时状态                      state.ts
  user.js               用户ID登录（★接 tt.login）      user.ts
  save.js               本地存档/校验/云同步预留         save.ts
  audio.js              程序化音效/静音                 audio.ts
  fx.js                 粒子/光束/飘字/屏震             fx.ts
  synergy.js            羁绊计算（纯函数）              synergy.ts
  fleet.js              单位工厂/敌军生成/站位          fleet.ts
  forge.js              熔炉物理/投放/合成/过载         forge.ts
  board.js              编队阵位/拖拽/升星              board.ts
  battle.js             自动战斗/战术技/胜负            battle.ts
  economy.js            金币消费/结算页广告入口         economy.ts
  ads.js                广告点位（★接抖音SDK）          ads.ts
  story.js              剧情队列状态机                  story.ts
  menu.js               登录/主界面/星图/设置           menu.ts
  render/ui/input/main  绘制/DOM/输入/装配              flow.ts + view/(Painter/GameApp)
                        —— 新增 ——                     tutorial.ts（新手指引）
                        —— Cocos 专有 ——               platform.ts（存储/广告/键盘三端适配）
                                                        view/Tex.ts（贴图库）
lib/matter.min.js       物理库（已内置；Cocos 版设为插件脚本）
```

## 想调什么改哪里

数值平衡 → `config.js`（TIERS/VALUE/DROP_POOL/STAR_MUL/费用）；羁绊效果 → `synergy.js`；敌军/Boss 强度 → `fleet.js` 的 `genEnemies()`；剧情台词 → `story-data.js`；广告对接 → `ads.js` / `platform.ts`；重力手感 → `forge.js` 的 `engine.gravity.y`；美术贴图 → `cocos/assets/resources/tex/` + `view/Tex.ts`。
