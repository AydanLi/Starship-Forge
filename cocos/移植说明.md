# 星舰熔炉 · Cocos Creator 移植说明

> 本目录（`cocos/`）是完整的 Cocos Creator 3.8 工程。全部玩法逻辑已翻译为 TypeScript 并通过 12 项无头回归测试（与 web 原型行为一致）；渲染层用 Graphics+Label 重写。你只需在编辑器里做一次 **3 分钟的初始化**（下面第 3 步），之后就能预览和构建抖音小游戏。

## 一、架构一览

```
cocos/
├── package.json / tsconfig.json      工程标识（Creator 打开后会自动补全 settings/temp 等）
├── assets/
│   ├── lib/matter.min.js             物理库（需勾选 Import As Plugin，见下）
│   └── scripts/
│       ├── globals.d.ts              全局声明（Matter / tt）
│       ├── core/  ← 纯逻辑，零 cc 依赖，与 web 原型 js/ 一一对应
│       │   platform.ts               ★平台适配：存储/激励视频/键盘（浏览器 & 抖音双实现）
│       │   config / storyData / state / user / save / audio / bgm / fx /
│       │   synergy / fleet / forge / board / battle / economy /
│       │   ads / story / menu / flow.ts
│       └── view/  ← 唯一 cc 相关层
│           Painter.ts                Graphics+Label池 画笔（canvas 2D 同形接口）
│           Bgm.ts                    背景音乐播放器（AudioSource，resources/audio/ 8 首）
│           GameApp.ts                ★唯一要挂到场景的组件（渲染/触摸/主循环）
```

要点：**改玩法数值仍然只改 core/config.ts**；接抖音登录只改 `user.ts`；广告位 ID 在 `platform.ts`（搜 `REPLACE_WITH_YOUR_AD_UNIT_ID`）。web 原型（仓库根目录）继续保留，作为快速试玩和逻辑参照。

## 二、安装 Cocos Creator（一次性）

1. 下载 **Cocos Dashboard**：https://www.cocos.com/creator-download
2. 在 Dashboard 里安装编辑器版本 **3.8.x**（任一 3.8 小版本均可）。
3. （抖音那步才需要）下载 **抖音开发者工具**：https://developer.open-douyin.com/ → 资源中心 → 开发者工具。

## 三、首次打开工程（约 3 分钟，一次性）

1. Dashboard → 项目 → **打开其他项目** → 选 `D:\Game project\星舰熔炉\cocos`。首次打开会生成 `library/ temp/ settings/` 等目录（已被 .gitignore 忽略），耐心等编译。
2. **把 matter.js 设为插件**：资源管理器（Assets 面板）选中 `lib/matter.min.js` → 右侧属性检查器勾选 **导入为插件（Import As Plugin）** 下的「允许 Web 平台加载」和「允许小游戏平台加载」→ 点右上 ✔ 应用。
3. **创建场景并挂组件**：
   - 菜单 文件 → 新建场景，Ctrl+S 保存为 `assets/main`（场景名 main）。
   - 层级管理器里应有 `Canvas` 节点（若无：右键 → 创建 → UI 组件 → Canvas）。
   - 选中 `Canvas`，属性检查器点 **添加组件 → 自定义脚本 → GameApp**。
4. **设计分辨率**：菜单 项目 → 项目设置 → 项目数据：设计宽度 **480**、设计高度 **840**，勾选 Fit Height（适配高度）。
5. 点顶部 **▶ 预览**（浏览器）。应看到星空登录页 → 点输入框（浏览器会弹 prompt）输入代号 → 进入主菜单 → 完整游戏。

> 若控制台有报错：直接把红字完整复制给我，我改完写回，你刷新即可。这是移植的正常调试环节。

## 四、构建抖音小游戏 + 真机预览

前置：抖音小游戏 **AppID**（在抖音开放平台 https://developer.open-douyin.com/ 注册小游戏应用获得，个人主体可用）。

1. Creator 菜单 项目 → **构建发布**：平台选 **抖音小游戏（bytedance-mini-game）**，填 AppID，构建。
2. 产物在 `cocos/build/bytedance-mini-game/`。
3. 打开 **抖音开发者工具** → 导入项目 → 选上面的构建目录 → 登录你的开放平台账号。
4. 工具里点 **预览** 生成二维码 → 用抖音 App 扫码 → **真机运行**。
5. 提审前替换两处占位：
   - `platform.ts` 里的激励视频 **adUnitId**（开放平台 → 流量主 → 创建激励视频广告位）；
   - `user.ts` 的登录换 `tt.login`（现在的“代号登录”真机也能玩，只是没绑 openid）。
