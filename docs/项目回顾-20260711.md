# 星舰熔炉 · 项目回顾(2026-07-11)

> 范围:立项至今的全量回顾——已完成里程碑、未来设计、技术债台账、上线关键路径。
> 结论先行:**代码侧健康、无架构硬伤,距上线只差 M5 一个里程碑;真正的瓶颈风险在两件行政事项(软著、抖音 AppID)。**

## 1. 进展:路线图五个里程碑已完成四个

### M0 玩法与架构基座 ✅

熔炉合成(merge-drop)× 自走棋对战的完整循环跑通,从 web 原型移植 Cocos Creator 3.8 落地。最有价值的决策是 **core 零 cc 依赖的分层**:

```
数据层   config / storyData              纯数据,零依赖
服务层   rng / platform / save / user / audio
系统层   state(G) ← forge / board / battle / synergy / fleet / economy / ads / story / tutorial / bgm
装配层   flow.ts                          状态机 + 输入路由 + bind 回调接线
视图层   view/  GameApp + Painter + Tex / Bgm
```

纯 TS 逻辑层 + 唯一触 cc 的视图层,让后续每一步规则改动都有无头测试兜底。

### M1 确定性与资源基建 ✅

- 可播种 RNG(mulberry32,seed 入档):复盘、测试、每日挑战共用一个地基
- vitest 无头回归体系(Matter.js 桩),现 **60 项**全绿
- 8 首即梦 AI BGM 以 Asset Bundle 方式入工程(懒加载、缺曲静音回退)

### M2 跨波舰队 ✅

刚体回流(存活舰带标签/星级回炉)、残骸回收(VALUE×星级×0.3)、船坞上限(10 艘,超限 0.6 折价)、失败重试原样还原、熔炉快照入存档。**舰队从「每波一次性」变成跨波资产**,这是留存设计的核心;过载线因此成为天然的舰队规模上限。敌方强度全部数据化(config.ENEMY)。

### M3 构筑可控 ✅

标签软权重(已成型方向 ×2.0 概率)+ 三选一定向招募(补羁绊缺口保底,15💰 或看广告免费)。玩家对成型路线有了掌控感,广告位也自然嵌入了玩法决策。

### M4 Boss 五机制 ✅

五个星区 Boss 各有一个围绕「战术技何时按」的机制:S1 半血召唤 / S2 周期护盾 / S3 孵化虫群 / S4 信号压制 / S5 锁定齐射,施放前 1 秒黄圈预警。**全部数据化在 `config.BOSS_SKILLS`,通用状态机解释执行**,调平衡不碰逻辑。

### 穿插完成的大项

- **全量架构审视**(详见 docs/架构审视报告.md):修掉 3 个帧率/时钟类真 bug——物理速度耦合刷新率(120Hz 真机 2 倍速)、setTimeout 逃逸游戏循环、core 内 performance.now;附带粒子 dt 缩放、Painter 颜色缓存

## 3. 技术债台账(全部在案,无失控项)

| 债务 | 危害 | 还债时机 |
|---|---|---|
| 存档单键覆盖写 | 写入途中进程被杀理论可丢档 | **M5** 接 tt 存储时加双缓冲(key_A/B+代数),~20 行 |
| GameApp 渲染错误只报一次后静默吞 | 真机黑屏无线索 | **M5** 埋点时一并上报 render error |
| audio.ts 在 core 内摸 WebAudio | 违反「只有 platform 感知环境」,已豁免的唯一例外 | **M5** 换真实音效文件接 AudioSource 时自然消除;此前不得新增第二个例外 |
| flow.ts per-phase switch 长毛 | 每加一个面板塞一处特判 | **下一个新面板出现时**重构为 phase→handler 表 + panel 栈 |
| synergy 手写 if 链 | 第 5 阵营时改动面大 | 加第 5 阵营时改数据表驱动 |
| 单条随机流 | 每日挑战「操作分叉掉落」(roguelike 常态,可接受) | 若要求当日掉落全网严格一致,拆 drop/combat 双流 |
| 模块级单例(forge engine/ads/bgm) | 同进程无法跑双对局(ghost 回放/数值模拟器) | 不改;真要模拟器时抽 battle+synergy+fleet 为可实例化 sim 包 |

| UI 几何混在 core(SLOTC/RC/BTN_*) | 「命中区共享」模式的代价,可接受 | 做多分辨率时统一收拢 C.UI(当前不投入) |
